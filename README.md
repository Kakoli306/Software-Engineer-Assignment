## Installation

- git clone https://github.com/Kakoli306/Software-Engineer-Assignment

- cd Software-Engineer-Assignment
- composer install
- cp .env.example .env
- Update .env and set your database credentials
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- npm install
- npm run watch
- php artisan serve
